# devArticles

A curated collection of technical articles, tutorials, and deep dives by **MicroClub USTHB**.

## Project Structure
- **/articles**: Categorized technical content (AI, Backend, Frontend, DevOps, etc.).
- **/assets**: Diagrams and images used in the articles.
- **/templates**: Standardized templates for new contributions.

## Contributing
Want to share your knowledge? 
1. Choose a category in `articles/`.
2. Use the [article-template.md](templates/article-template.md) to draft your post.
3. Submit a Pull Request.

## Platforms
Our articles are shared on platforms like **Medium** and social media to help the tech community grow.
